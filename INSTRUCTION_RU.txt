3. КАК ПОЛЬЗОВАТЬСЯ

ОБРАБОТКА ОДНОГО ФАЙЛА:
   1. Нажмите большую синюю кнопку "Выбрать PDF"
   2. Выберите файл с жалобой гражданина
   3. Программа автоматически:
      - Извлечет данные из PDF
      - Просклоняет имена в дательный падеж

   4. Word-файл появится в папке "output"

ОБРАБОТКА НЕСКОЛЬКИХ ФАЙЛОВ (drag & drop):
   1. Выделите несколько PDF-файлов в проводнике
   2. Перетащите их мышкой в окно программы (в серую зону)
   3. Программа обработает все файлы автоматически

6. ВОЗМОЖНЫЕ ПРОБЛЕМЫ И РЕШЕНИЯ
═══════════════════════════════════════════════════════════════════

[ERROR] "Windows защитила ваш компьютер"
   → Нажмите "Подробнее" → "Выполнить в любом случае"
   (Это нормально для неподписанных программ)

[ERROR] Программа не запускается
   → Проверьте путь к папке (только английские буквы!)
   → Убедитесь, что не блокируется антивирусом
   → Попробуйте "Запуск от имени администратора"

[ERROR] Word-документы не создаются
   → Проверьте, что папка "templates" содержит .docx файлы
   → Убедитесь, что у вас есть права на запись в папку "output"

[ERROR] Ошибки парсинга PDF
   → Убедитесь, что PDF соответствует формату жалоб ОАТИ
   → Проверьте кодировку текста (должна быть UTF-8)

9. МОДУЛЬ ЭЗП (ЭЛЕКТРОННЫЕ ЗАКАЗНЫЕ ПИСЬМА) - v4.2
═══════════════════════════════════════════════════════════════════

🎯 БЫСТРЫЙ СТАРТ:

   1. Перейдите на вкладку "Работа с ЭЗП"
   2. Бросьте PDF файлы в зону Drag & Drop
   3. Нажмите "Обработать ЭЗП"
   4. Готово! Excel файл появится в папке output/

📋 НОВОЕ В v4.2 - УПРОЩЕННЫЙ WORKFLOW:

   ✓ Больше НЕ НУЖНО загружать Excel вручную!
   ✓ Программа автоматически использует шаблон
   ✓ Каждая обработка создает новый файл
   ✓ Результаты в output/Отправка_YYYYMMDD_HHMMSS.xlsx

КАК РАБОТАЕТ:

   ШАГ 1: Подготовка PDF
      - Убедитесь, что PDF содержат адреса в верхнем правом углу
      - Файлы могут быть любого размера

   ШАГ 2: Загрузка файлов
      - Перетащите PDF в зону "Перетащите PDF файлы сюда"
      - ИЛИ кликните на зону и выберите папку с PDF
      - Программа покажет количество найденных файлов

   ШАГ 3: Обработка
      - Нажмите кнопку "Обработать ЭЗП (AI)"
      - Программа автоматически:
        * Извлечет адреса из PDF
        * Определит тип получателя (юр/физ лицо)
        * Заполнит Excel таблицу по формату Почты России
        * Сохранит результат в output/

   ШАГ 4: Получение результата
      - Найдите файл в output/Отправка_ДАТА_ВРЕМЯ.xlsx
      - Откройте его в Excel
      - Готово к отправке на Почту России!

🤖 AI ИНТЕГРАЦИЯ (ОПЦИОНАЛЬНО):

   Программа может использовать YandexGPT для:
   - Автоматического определения юр/физ лица
   - Повышения точности классификации адресов

   Настройка AI:
   1. См. файл YANDEX_API_SETUP.txt
   2. Получите API ключ в Yandex Cloud
   3. Настройте через вкладку "AI Помощник"

   Работа БЕЗ AI:
   - Программа работает и без AI!
   - Используется встроенный rule-based метод
   - Точность ~85-90%

📁 СТРУКТУРА ФАЙЛОВ ЭЗП:

   templates/ezp_template.xls   ← Базовый шаблон (НЕ ТРОГАТЬ!)
   output/Отправка_*.xlsx       ← Результаты обработки

ВАЖНО:
   • Шаблон templates/ezp_template.xls должен существовать
   • Не редактируйте шаблон вручную
   • Каждая обработка создает НОВЫЙ файл
   • Старые результаты НЕ перезаписываются

10. AI ПОМОЩНИК (ОПЦИОНАЛЬНО)
═══════════════════════════════════════════════════════════════════

Вкладка "AI Помощник" позволяет:
   ✓ Генерировать тексты уведомлений
   ✓ Создавать union-параграфы
   ✓ Улучшать определение типа получателя в ЭЗП

НАСТРОЙКА AI:
   1. Откройте YANDEX_API_SETUP.txt
   2. Следуйте инструкциям по получению ключа
   3. Нажмите "Настроить AI" во вкладке
   4. Введите Folder ID и API Key
   5. Сохраните

ПРОВЕРКА РАБОТЫ:
   - Индикатор статуса:  🟢 Online - работает
                        🔴 Offline - не настроено
   - Кнопка "Тестировать AI" проверяет подключение

ИСПОЛЬЗОВАНИЕ:
   - AI автоматически используется в ЭЗП
   - Ручной ввод текста для генерации
   - Копирование результатов в буфер обмена
