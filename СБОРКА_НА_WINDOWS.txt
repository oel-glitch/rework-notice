=====================================

Версия: v4.4 + ЭЗП rewrite от 19.04.2026
Исправления:
  - Переписан AddressParser (координатный разбор, а не поиск жирного текста)

  - Шаблон ezp_template.xlsx (старый .xls не читался openpyxl)


ЧТО ДЕЛАТЬ
----------

1. Скопировать всю эту папку на Windows-машину в любое удобное место.
   (Например: C:\Users\<ты>\Documents\auto-notice\)

2. Открыть проводник → в этой папке дважды кликнуть по файлу:

      build.bat

3. Откроется консоль. Скрипт сам:
   - проверит/скачает переносной Python 3.11 (~13 МБ)
   - установит все зависимости из requirements.txt
   - соберёт .exe через PyInstaller
   - упакует результат в архив PDF_Parser_OATI_Standalone.zip

   Время сборки: 2-5 минут (в зависимости от интернета).

4. Готовый архив окажется рядом — PDF_Parser_OATI_Standalone.zip.
   Его можно распаковать и запустить PDF_Parser_OATI.exe.


ЕСЛИ ЧТО-ТО ПОШЛО НЕ ТАК
------------------------

- "python не является внутренней командой": скрипт скачает portable
  Python автоматически, НО нужен интернет. Если интернета нет —
  предварительно установить Python 3.11 вручную с python.org.

- "Failed to install dependencies": проверить интернет, попробовать
  перезапустить build.bat. Порт 443 (https) должен быть открыт.

- Ошибка при сборке — см. файл BUILD_GUIDE_RU.txt (подробности).


ПРОВЕРКА ПАРСЕРА (опционально, до сборки)
------------------------------------------

Можно прогнать автотесты прежде чем собирать:

    pip install -r requirements.txt
    pytest tests/test_ezp_address_parser.py -v

Должно быть "16 passed".


ФАЙЛЫ В ЭТОЙ ПАПКЕ
------------------

  build.bat                 — запустить для сборки
  build.py                  — внутренний скрипт сборки (вызывается из build.bat)
  main.py                   — точка входа GUI
  src/                      — исходный код (ezp_processor.py и др.)
  templates/                — Word-шаблоны уведомлений + ezp_template.xlsx
  config/, database/        — конфиги и пустая БД
  requirements.txt          — список зависимостей
  tests/                    — автотесты и фикстуры PDF
  INSTRUCTION_RU.txt        — инструкция пользователя готового .exe
  PDF_Parser_OATI.spec      — запасная спека PyInstaller (build.py её не использует)
