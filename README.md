# PDF Парсер и Генератор Word Документов
**Версия 4.2** - Автоматический парсинг PDF обращений граждан, генерация уведомлений в Word с правильным склонением имен, упрощенная обработка ЭЗП и AI-интеграция.

## Возможности

### Основные функции
- 📄 **Автоматический парсинг PDF**: Извлечение ФИО, email, дат, номеров обращений, резолюций
- 🔄 **Склонение русских имен**: Автоматическое склонение в дательный падеж с определением рода
- 📝 **Генерация Word документов**: Создание уведомлений по шаблонам с правильным наименованием файлов
- 💾 **Локальная база данных**: SQLite для хранения справочников и истории обработки
- 🚀 **Массовая обработка**: Обработка нескольких PDF файлов за один запуск
- 🖱️ **Drag & Drop**: Удобный интерфейс с поддержкой перетаскивания файлов
- 🔒 **Offline-First**: Все чувствительные данные обрабатываются локально

## Формат входных данных (PDF)

PDF должен содержать:
- ФИО гражданина
- Email адрес
- ID портала (идентификатор обращения)
- Номер ОАТИ (формат: 01-21-П-XXXX/25)
- Дата регистрации
- Резолюция с указанием части закона (ч.3 или ч.4 ст.8 ФЗ №59-ФЗ)
- Департаменты/инспекции для направления

## Формат выходных файлов (Word)

Имя файла: `Уведомление_гражданина_Фамилия_И_О_XXXX_25.docx`

Где:
- `Фамилия` - фамилия гражданина
- `И` - первая буква имени
- `О` - первая буква отчества
- `XXXX` - центральные 4 цифры из номера ОАТИ
- `25` - год

Выбор шаблона:
- **ч.3** - один департамент → `template_ch3.docx`
- **ч.4 + несколько департаментов** → `template_ch4_multiple.docx`
- **ч.4 + префектура** → `template_ch4_prefecture.docx`

## Возможные проблемы и решения

### Приложение не запускается

- Убедитесь, что используется 64-bit версия Windows
- Попробуйте запустить от имени администратора
- Проверьте, что антивирус не блокирует .exe файл
- Убедитесь, что на диске достаточно места (минимум 500 MB)

### Шаблоны не извлекаются из .exe

- При первом запуске появится предупреждение если шаблоны не найдены
- Проверьте, что .exe файл не поврежден (попробуйте скачать заново)
- Убедитесь, что у приложения есть права на создание папок
- В крайнем случае, создайте папку `templates/` вручную и добавьте .docx шаблоны

### Ошибка при парсинге PDF

- Проверьте формат PDF (должен быть текстовый, не отсканированный)
- Убедитесь, что PDF содержит все необходимые поля
- Проверьте кодировку (должна быть UTF-8)

### Неправильное склонение имен

- Проверьте правильность написания имени в исходном PDF
- Для редких имен может потребоваться ручная корректировка
- Убедитесь, что отчество указано (для определения рода)

### Drag & Drop не работает

- В некоторых средах (например, Replit) Drag & Drop недоступен
- Используйте кнопку "Выбрать PDF файлы"
